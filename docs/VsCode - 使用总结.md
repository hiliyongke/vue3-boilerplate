1.  Auto Reanem Tag：修改前后标签。
2.  Prettier：ctrl+s 保存格式化代码。要设置`format on save`打勾。
3.  Beautify：输入指令格式化代码。
4.  Bracket Pair Colorizer2：括号对着色。
5.  Browser Preview：浏览器预览，右边会多一个浏览器的图标，相当于在右边打开 Chrom。当然，两个屏幕就不用安装这个插件了。
6.  Community Material Theme：在插件栏中设置这个插件`set color theme`，就可以设置不同的颜色了，另一个主题插件：Ayu。
7.  indent-rainbow：缩进着色。
8.  LeetCode：百度 “力扣”（刷题用的）。安装完这个插件之后，左边会多一个力扣的图标，使用力扣账户登录，就可以看题库了，关于解决方法、讨论等都有。题目的右下角会有一个`Code New`，然后`Just open the problem file`，就可以右边是描述，左边是解决方案。
9.  Live Server：服务器模式预览代码。HTML 右键可以`Open with Live Server`。代码保存之后会实时变化。如果移动端需要做真机调试的时候有用。
10. WakaTime：编程统计时间的工具，这个插件可以在它的网页端查看你的代码编写情况，该插件支持很多编辑器，包括 WebStorm 等等。
11. Chinese：中文语言包。
12. Vetur：Vue 的支持插件。
13. EMMET：HTML 的快速编辑，官网有文档。
14. CSS Peek：标签中的 CSS 跳转。
15. color highlight：16 进制颜色预览插件。可以在设置中 UserSettings>>Extensions>>Color Highlight>>Maker Type 设置预览模式。
16. Code Runner：java、js 等文件右键就可以运行了。
17. gitignore：git 忽略文件插件，文件右键就可以 Add to gitignore。
18. import Cost：文件分析插件，js 依赖项的大小。
19. Paste Image：编辑 Makrdown 文件时，快速粘贴复制的图片。
20. project mananger：项目管理插件，安装好之后左边就会有一个项目管理的图标。在打开的项目中，按 Ctrl+Shfit+P 调出命令行面板。这里就会有保存项目。这个项目就可以保存在左边的图标中，之后可以方便打开。
21. vscode-icons：文件图标。
22. Documemt This：注释插件，在方法上按 Ctrl+Alt+D 按两次就可以根据方法生成注释。
23. Language PL/SQL：支持 SQL，编辑 SQL 文件用的。
24. GitLens：[Git 的深入扩展](https://blog.l0v0.com/posts/a91a4c58.html)。

卸载：卸载软件之后，删除：`C:\Users\xxx\.vscode`与`C:\Users\xxx\AppData\Roasing\Code`。
